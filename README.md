# 🏢 BDS — Business Development System

![Katalon Studio](https://img.shields.io/badge/Katalon%20Studio-Latest-green)
![Platform](https://img.shields.io/badge/Platform-Web-blue)
![Type](https://img.shields.io/badge/Type-Manual%20%26%20Automation-orange)
![Status](https://img.shields.io/badge/Status-Active-brightgreen)

Repositori ini berisi **Test Suite** untuk aplikasi **BDS (Business Development System)**, sebuah platform bisnis internal berbasis web. Pengujian mencakup aktivitas **Manual & Automation Testing** menggunakan **Katalon Studio** untuk memastikan seluruh alur bisnis berjalan dengan benar dan sesuai ekspektasi.

## 🔍 Key Scope

* ✅ **Functional & Regression Testing** — Validasi fitur dan deteksi regresi pasca perubahan.
* 🔄 **Business Flow Validation** — Memastikan alur bisnis end-to-end berjalan sesuai ekspektasi.
* 🗄️ **Data Verification** — Verifikasi keakuratan dan konsistensi data pada sistem.
* 🐛 **Defect Reporting & Tracking** — Pencatatan dan pelacakan bug dari penemuan hingga resolusi.

---

## ✨ Fitur Unggulan

* 🌐 **Web-Based Testing** — Pengujian berbasis browser menggunakan Katalon Studio.
* 🔁 **Reusable Keywords** — Custom keywords yang dapat digunakan ulang di berbagai Test Case.
* 📂 **Modular Test Structure** — Test Case diorganisir per modul bisnis untuk kemudahan navigasi.
* 📊 **Auto Report Generation** — Laporan hasil eksekusi otomatis tersimpan di folder `Reports/`.
* 🗃️ **Data-Driven Testing** — Mendukung input data dari file CSV/Excel untuk pengujian berlapis.

## 📂 Struktur Proyek

```text
BDS-HOST-TESTING/
│
├── Test Cases/               # Script pengujian per modul bisnis
├── Test Suites/              # Kumpulan test suite & collection
├── Keywords/                 # Custom keywords & reusable logic
├── Object Repository/        # Web element locators
├── Data Files/               # Test data (CSV, Excel)
├── Reports/                  # Hasil eksekusi test

│   ├── INSTALLATION.md
│   ├── USER_GUIDE.md

```
